# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-05-02

- First Release Candidate.
- Sleek, exquisite dark mode interface with collapsible settings and status dots.
- Asynchronous MediaInfo metadata extraction with background deep scan fallback.
- Advanced track extractions (Video, Audio, Extra) with proper percentages and math.
- Robust Windows execution with output captured to log file and no popped up terminals.
